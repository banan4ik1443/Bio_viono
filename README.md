# Iris: Био-войны — игровой бот (Telegram)

Бот повторяет механику оригинальной мини-игры «Био-войны» (лаборатория,
навыки, заражение, пассивный доход с жертв, топы) плюс собственные команды
**фарм** и **мешок** для добычи и просмотра валюты. Корпорации не реализованы.

Код разделён на два файла: `main.py` (вся логика) и `config.py` (все
настройки — токен, баланс игры, слова-триггеры команд).

## Установка

1. Установи Python 3.10+
2. Установи зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Получи токен бота у [@BotFather](https://t.me/BotFather) в Telegram
4. Открой `config.py` и вставь токен в переменную `BOT_TOKEN`
5. Запусти бота:
   ```
   python main.py
   ```

## Все команды

Команды пишутся текстом в чат, с "!" впереди или без него, регистр не важен.

### Фарм
- `фарм` / `фарма` — фармить койны, кулдаун 5 минут
- `мешок` — посмотреть баланс

### Лаборатория
- `!лаб` (синонимы: `моя лаборатория`, `моя лаба`, `моя лаб`) — профиль лаборатории.
  Создаётся автоматически при первом вызове
- `!лаб в лс` — то же самое, но в личку
- `+лаб` / `-лаб` — разрешить / запретить другим смотреть твою лабу
- `+имя лабы {название}` / `-имя лабы`
- `+имя патогена {название}` / `-имя патогена`

### Прокачка навыков
- `+патоген {N}` — ячейки хранения патогенов
- `+разработка {N}` — увеличивает пассивный доход с заражённых жертв
- `+заразность {N}` — усиливает шанс заражения
- `+иммунитет {N}` — защита от заражения
- `+летальность {N}`
- `+безопасность {N}`

Максимум 5 уровней за раз. Цена растёт с текущим уровнем навыка.
`N` можно не указывать — тогда прокачка на 1 уровень.

### Удаление / восстановление
- `-биоигра` (синоним `-игра вирусы`) — удалить лабораторию
- `!восстановить лабу` — восстановить со штрафом характеристик и потерей жертв

### Заражение и пассивный доход
- `заразить` (ответом на сообщение игрока) — заразить конкретного человека
- `заразить {число}` (ответом на сообщение) — несколько попыток подряд
- `заразить слабее` / `заразить сильнее` / `заразить равный` — заразить случайного
  игрока чата слабее/сильнее/равного по прокачке
- `заразить чат` — заразить случайного участника чата
- `заразить рандом` (сокращение `заразить р`) — заразить случайного игрока всей игры
- `заразить рандом слабее/сильнее/равный` — то же с фильтром по силе
- `вакцина` — снять горячку за койны (также лечит от активного заражения)
- `доход` (синоним `мой доход`) — посмотреть, сколько активных жертв и сколько
  дохода они приносят за один тик

**Как работает пассивный доход:** пока жертва находится в горячке после
успешного заражения, бот раз в `PASSIVE_INCOME_INTERVAL_MINUTES` (по умолчанию
10 минут) начисляет атакующему доход за каждую активную жертву и присылает
уведомление в личные сообщения. Доход растёт с уровнем навыка «Разработка».
Как только горячка у жертвы проходит (или она лечится вакциной), заражение
снимается и доход по этой жертве перестаёт начисляться.

⚠️ Чтобы получать уведомления о доходе в личку, нужно один раз написать боту
`/start` в личном чате — иначе Telegram не разрешает боту писать первым.

### Топы
- `биотоп` — глобальный топ по био-опыту
- `биотоп чата` — топ по опыту среди участников этого чата
- `биотоп заражений` — топ по количеству заражённых жертв

### Прочее
- `/start` — приветствие и краткий список команд (единственная команда со слэшем)
- `вирус помощь` (синонимы: `био помощь`, `биоатака помощь`) — ссылка на справку

## Настройка баланса игры

Все параметры собраны в файле `config.py`:

- `FARM_COOLDOWN_MINUTES`, `FARM_BASE_REWARD`, `YIELD_MIN/MAX`,
  `BONUS_MIN/MAX_PERCENT` — параметры фарма
- `CURRENCY_NAME`, `CURRENCY_SYMBOL` — название и символ валюты (сейчас "койны" / `i¢`)
- `SKILL_BASE_COST`, `SKILL_COST_PER_LEVEL`, `MAX_LEVELS_PER_UPGRADE`,
  `EXP_PER_LEVEL` — стоимость и лимиты прокачки навыков
- `INFECTION_BASE_CHANCE`, `FEVER_BASE_MINUTES`,
  `FEVER_MINUTES_PER_ATTACKER_LEVEL`, `VACCINE_COST` — баланс заражений
- `PASSIVE_INCOME_ENABLED`, `PASSIVE_INCOME_INTERVAL_MINUTES`,
  `PASSIVE_INCOME_PER_VICTIM_BASE`, `PASSIVE_INCOME_PER_DEVELOPMENT_LEVEL` —
  баланс пассивного дохода с жертв
- `RESTORE_PENALTY_MULTIPLIER` — штраф при восстановлении лабы

Там же, в config.py, в блоке "ТРИГГЕРЫ ТЕКСТОВЫХ КОМАНД", можно добавлять
синонимы для любой команды — просто впиши новое слово в нужный набор,
например:
```python
FARM_TRIGGERS = {"фарм", "фарма", "farm", "урожай"}
```

## Данные

Прогресс хранится в SQLite-файле `data/iris_bot.db`. Папка `data`
создаётся автоматически при первом запуске бота. Не удаляй эту папку —
иначе весь прогресс игроков потеряется.

## Добавление в группу

1. Добавь бота в чат
2. Дай ему права администратора ИЛИ отключи Privacy Mode у @BotFather
   (`/setprivacy` → Disable), иначе бот не увидит сообщения в группе
   без упоминания или реплая

## Известные ограничения

- Команда `заразить {ссылка}` работает через **реплай** на сообщение цели
  (упоминание по `@username` без реплая Telegram не даёт боту напрямую
  узнать user_id без дополнительных прав)
- Корпорации не реализованы
- Уведомления о доходе приходят только тем, кто хотя бы раз писал боту в личку
